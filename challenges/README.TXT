📁 challenges/

This folder contains SQL challenge sets and their detailed solutions.

Each file corresponds to a major chapter in the book and includes:
- A short real-world scenario
- Problem statement
- SQL query
- Expected output (column names and sample values)

Designed to test your knowledge and reinforce learning after each chapter:
- Challenge 1: Basic queries
- Challenge 2: Aggregations
- Challenge 3: Funnel analysis
- Challenge 4: Segmentation
- Challenge 5: A/B testing & conversions

💡 Use these challenges to benchmark your progress or run a team quiz.
